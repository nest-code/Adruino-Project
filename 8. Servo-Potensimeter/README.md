# Membuat Project Arduino - Servo-potensimeter

List :
- 1 Arduino Uno
- 1 Breadbord
- 1 Servo ( Micro Servo 9g /SG(SG90)
- 1 Potensiometer (B10K)
- 8 Kabel Jumper

Explanation:

Servo mempunyai 3 kabel 
- Jingga (Orange) = Pin Inputan (Pin)
- Merah (Red) = Volt (5V)
- Coklat (Brown) = Ground

Potensiometer
- Volt (5v)
- Analog
- Ground
